#### 安装依赖

```
npm install
```

#### 执行全量测试

```
npm run test
```

#### 将jest 全局安装 
可进行 单元测试
```
npm install jest --global
```

#### 单个文件测试


```
Foo.js(文件名) 
npm test -- Foo
```
